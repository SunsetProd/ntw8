# Network 8

Статичный сайт для подготовки к финальному тесту по сетям.

## Запуск локально

```bash
python3 -m http.server 8000
```

Открыть: http://127.0.0.1:8000/

## GitHub Pages

Сайт работает без сборки: входная точка `index.html`, стили `styles.css`, логика `app.js`.
Текст лекций встроен в `lectures-data.js`, поэтому папки с лекциями можно не загружать на GitHub Pages.
Скриншоты и исходный docx итогового теста не публикуются: на сайте оставлен только текстовый интерактивный тест.
В настройках репозитория выбери Pages из ветки и корня проекта.
